# ML Starter Template

A reproducible machine learning project template using Python, scikit-learn, and pandas.

## Structure
- `src/` : source code
- `tests/` : unit tests
- `notebooks/` : Jupyter notebooks
- `docs/` : documentation

## Quickstart
```bash
make install
make lint
make test
